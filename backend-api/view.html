HMCTS To Do List API

  • /api/tasks

    • GET

      Response: an array of all task objects:

      {
  "tasks": [
    {
      "task_id": integer,
      "title": string,
      "description": optional string,
      "status": string ("TODO" || "IN PROGRESS" || "COMPLETED")
      "due_date": date
    },
    ...
  ]
}

    • POST

      Request: accepts a task object as payload:

      {
    "title": string,
    "description": optional string,
    "status": string ("TODO" || "IN PROGRESS" || "COMPLETED")
    "due_date": date (YYYY-MM-DD)

}

      Response: responds with the newly created entry.

  • /api/tasks/:id

    • GET

      Response: returns a task object at a parametrics task_id:

      {
  "task":
    {
      "task_id": integer,
      "title": string,
      "description": optional string,
      "status": string ("TODO" || "IN PROGRESS" || "COMPLETED")
      "due_date": date
    }
}

    • PATCH

      Request: accepts an object as payload as with any of the following properties: 

      {
    "title": string,
    "description": string,
    "status": string ("TODO" || "IN PROGRESS" || "COMPLETED")
    "due_date": date (YYYY-MM-DD)

}

      Response: responds with the newly updated db entry.

    • DELETE

      removes a specified property from the database

      Response: none
